# Micro-copy Document
## Complete UI Copy with Guilt-Free Language

---

## Guiding Principles

1. **Never shame, blame, or judge**
2. **Be warm but not cheesy**
3. **Inform without overwhelming**
4. **Empower, don't lecture**
5. **Be concise (mobile screens are small)**

## Writing Style Guide

### Do

- Use contractions (you're, we'll, it's)
- Address user as "you"
- Use "we" for app actions
- Be specific and actionable
- Acknowledge user effort
- Frame suggestions positively

### Don't

- Use "please" excessively (sounds servile)
- Use exclamation points excessively
- Use jargon (TDEE, macros without context)
- Blame or shame
- Use urgent or alarming language
- Make assumptions about user goals

### Word Choice

| Avoid | Use Instead |
|-------|-------------|
| Failed | Didn't go through |
| Error | Something went wrong |
| Invalid | Doesn't look right |
| Required | Needed |
| Must | Need to |
| Cheat meal | (don't use) |
| Bad food | (don't use) |
| Guilty pleasure | (don't use) |
| You should | You could, Consider |
